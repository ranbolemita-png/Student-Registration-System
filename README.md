# MTU Student Registration System

A modern student registration system for Mizan-Tepi University built with HTML, Tailwind CSS, JavaScript, and Node.js.

## Features
* **Student Registration:** Simple form for students to enter their details.
* **Database Support:** Data export to CSV and JSON formats.
* **Admin Dashboard:** View and manage registered student lists.
* **Mobile Responsive:** Works perfectly on both desktop and mobile devices.

## Tech Stack
* **Frontend:** HTML5, Tailwind CSS, JavaScript
* **Backend:** Node.js
* **Storage:** CSV / JSON (Moving to MySQL soon!)
